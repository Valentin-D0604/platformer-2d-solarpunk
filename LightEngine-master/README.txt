To use the project you need to : 
1 - download the zip file and unzip it
2 - use "make.bat" to create the solution of the project
3 - click on the "ide" file, go to "vs" and then start LightEngine.sln

To play :
use "zqsd" to move, right click to shoot, left to parry, and E to dash. Space is used to jump

it's recommended to use a playstation controller to have more precise animation and movement
